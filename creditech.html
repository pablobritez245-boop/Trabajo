CrediTech

Sistema de gestión de créditos desarrollado como proyecto académico.

Descripción del proyecto

CrediTech es un sistema diseñado para la administración de créditos. Permite gestionar clientes, solicitudes de crédito, pagos y registros relacionados con operaciones financieras básicas.

El proyecto fue desarrollado utilizando HTML5 y Bootstrap, aplicando una estructura organizada, navegación entre páginas y una interfaz clara para facilitar el uso del sistema.

Volver a proyectos

Módulos del sistema

Clientes

Permite registrar y administrar la información de los clientes.

Créditos

Gestiona solicitudes de crédito y datos relacionados al préstamo.

Pagos

Registra pagos realizados y permite consultar información de cobros.
